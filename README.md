# EMBER

EMBER is a 2D action game built with Python and Pygame, inspired by Megabonk.

## Requirements

* Python 3
* pip
* Pygame
* NumPy

## Clone the Repository

```bash
git clone git@github.com:Aditya-as-rgb/EMBER.git
cd EMBER
```

Alternatively, using HTTPS:

```bash
git clone https://github.com/Aditya-as-rgb/EMBER.git
cd EMBER
```

## Install and Upgrade pip (Linux)

Update package lists and ensure Python and pip are installed:

```bash
sudo apt update
sudo apt install python3 python3-pip
```

Upgrade pip:

```bash
python3 -m pip install --upgrade pip
```

## Install Dependencies

Install the required Python packages:

```bash
pip install pygame numpy
```

If your system uses `pip3`:

```bash
pip3 install pygame numpy
```

## Run the Game

Navigate to the source directory:

```bash
cd src
```

Launch the game:

```bash
python3 main.py
```

## Controls

Refer to the in-game instructions for controls and gameplay.
